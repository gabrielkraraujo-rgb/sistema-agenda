Agenda — Guia de configuração

Somente o que depende de você. Clique em cada passo ao concluir — o progresso fica salvo neste navegador.

A parte técnica (instalar dependências, subir o banco, criar tabelas, rodar o sistema) é com o Claude Code: abra a pasta nele e peça "configure e rode este projeto" — ele segue as instruções que já estão no repositório. Esta lista é o que ele não pode fazer por você: contas, credenciais e autorizações.

Progresso da configuração 0 de 0

Começando

Extrair a pasta do projeto e abrir no Claude Code
Descompacte, abra a pasta no Claude Code e peça para configurar e rodar o projeto.
Docker Desktop instalado e aberto
O banco de dados local roda nele. Baixe em docker.com se não tiver — a instalação pede permissões que só você pode dar.
Decidir seu e-mail e senha de login
São os campos SEED_EMAIL e SEED_PASSWORD (mínimo 8 caracteres) do arquivo .env — o Claude preenche o arquivo, mas os valores são seus. É com eles que você entra no sistema.

Google Agenda conta Google

Criar a credencial OAuth no Google Cloud
No Google Cloud Console, logado na SUA conta: crie um projeto → habilite a Google Calendar API → "Tela de consentimento OAuth" (modo teste; adicione seu próprio Gmail como test user) → "Credenciais" → "ID do cliente OAuth", tipo Aplicativo da Web.
Redirect URI:http://localhost:3000/api/oauth/google/callback
Passar o Client ID e o Client Secret para o .env
Campos GOOGLE_CLIENT_ID e GOOGLE_CLIENT_SECRET. São segredos da sua conta — cole você mesmo (ou dite para o Claude preencher e reiniciar o sistema).
Conectar e autorizar
No sistema: aba Agendas → "Conectar Google" → autorize com a sua conta. Depois defina o nome e a cor de cada agenda que apareceu (botão "Editar").
Opcional: Outlook
Mesmo processo no Azure Portal (App registration multitenant), redirect http://localhost:3000/api/oauth/outlook/callback, campos MS_CLIENT_ID e MS_CLIENT_SECRET no .env.

Tempo de carro e atraso Google Maps

Criar a chave do Maps
No mesmo projeto do Google Cloud (exige billing ativo na sua conta): habilite Routes API e Places API (New) → "Credenciais" → "Chave de API".
Colar a chave em Ajustes e preencher seu endereço
No sistema: Ajustes → Google Maps → cole a chave e salve. Depois Perfil → Endereço: digite e selecione uma sugestão do Google ("Endereço validado"). É a sua origem para calcular o tempo até cada compromisso.

Notificações no WhatsApp Evolution API

Configurar a Evolution API em Ajustes
Ajustes → WhatsApp: URL base da sua instância Evolution, nome da instância e chave de API. Número de destino: o campo específico ou o telefone do Perfil.
Enviar mensagem de teste
Botão "Enviar teste" na mesma tela. Chegando no seu WhatsApp, os 4 avisos estão ativos: resumo diário, lembrete antes do evento, convite novo e alerta de atraso — horários e antecedência configuráveis ali.

Finalização

Trocar a senha no Perfil
Depois do primeiro login, defina uma senha definitiva em Perfil → Trocar senha (a do .env fica registrada em arquivo).
Conferir o fluxo completo
Criar um evento com local pelo "+", arrastar no calendário para mudar o horário, conferir que apareceu no Google Agenda e abrir no celular (dá para instalar como app pelo menu do navegador).
